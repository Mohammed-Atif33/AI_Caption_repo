AI Caption Generator 

A simple AI-powered caption generator that creates captions based on user input using an API

IMPROTENT!!

API Key Setup
Important: You must create your own .env file and add your API key before running the project.
Example:
    OPENROUTER_API_KEY=YOUR_API_KEY
    (Without any spaces)


1-Features

Generate captions instantly using AI.
Clean and simple UI.
Fallback system if AI is not working

2-Tech Stack

Python (Flask).
HTML, CSS.
OpenRouter API (or any AI API)

3-Future Improvements

Add more AI models.
Improve UI design.
Add download/share options
